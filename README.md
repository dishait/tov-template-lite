## 特点 🐳

1. Vite 的
2. Vue3 的
3. 文件路由
4. Api 自动引入
5. 组件自动引入
6. VueUse 支持
7. Unocss 的
8. TypeScript 的
9. 暗黑模式支持
10. pinia 状态管理
11. pnpm 包管理器
12. 跳转进度条支持
13. 插件自动加载支持
14. Vitest 单元测试支持
15. 路径别名 `~` 支持
16. 打包时资源压缩支持
17. 生产环境自动移除开发日志
